# Discord Bot

## Setting up your project

### global dependencies

    sudo npm install -g nodemon

### bot

    1. cd bot
    2. npm init -y #init project
    3. npm install discord.js
    4. nodemon

### api
    1. cd api
    2. npm init -y #init project
    3. npm install mysql2
    4. nodemon


### dashboard
    1. cd bot-landing-page
    2. npm init -y #init project
    3. npm install mysql2
    4. nodemon
